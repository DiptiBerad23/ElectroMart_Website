# ⚡ ElectroMart – Smart Electronics E-Commerce Platform

ElectroMart is a full-stack **electronics e-commerce web application** built to deliver a smooth and user-friendly online shopping experience.  
The project showcases clean UI design and a well-structured backend using **Java and Spring Boot**.

---

## ✨ Features

- Browse electronic products  
- View product details  
- User login & registration  
- Add to cart functionality  
- Responsive and clean UI  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java, Spring Boot  
- **Database:** MySQL  
- **Tools:** Git, GitHub, STS
