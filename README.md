# ⚡ ElectroMart – Smart Electronics E-Commerce Platform

ElectroMart is a full-stack **electronics e-commerce web application** designed to provide a smooth, secure, and user-friendly shopping experience.  
It demonstrates clean UI design combined with a **robust Spring Boot backend**, following real-world development practices.

---

## ✨ Key Features

- 🛒 Browse a wide range of electronic products  
- 📄 View detailed product information  
- 👤 User registration & login  
- ➕ Add products to cart  
- 📱 Fully responsive and clean UI  

---

## 🛠️ Technology Stack

- **Frontend:** HTML, JSP, CSS, JavaScript  
- **Backend:** Java, Spring Boot  
- **Database:** MySQL  
- **Tools:** Git, GitHub, Spring Tool Suite (STS)  

---

⭐ *A simple, scalable project showcasing practical full-stack development skills.*
